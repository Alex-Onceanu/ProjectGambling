# Project *GAMBLING*

## Présentation

Un jeu de poker *en ligne* !

## But du jeu

C'est un jeu de Poker. Je vous renvoie donc aux règles classiques du Texas Hold'em. Le but est de bluffer et miser pour devenir riche et pouvoir ensuite invoquer plein de skins. 

## Comment jouer

Cliquez sur "JOUER", entrez votre pseudo et attendez que le serveur soit lancé.

Une fois en jeu, votre tour est indiqué par vos cartes qui s'agrandissent périodiquement. Vous pouvez miser à l'aide des boutons "suivre", "se coucher" et "surenchérir". Vous pouvez appuyer sur espace pour suivre, et sur échap pour mettre pause.

Une fois riche, rendez-vous dans la boutique (depuis le menu pause ou l'écran titre) pour invoquer des nouveaux skins ! Ils sont de différentes raretés (rare, épique ou légendaire), il y en a une vingtaine. Pour équiper un skin, sélectionnez-le dans le menu à droite dans la boutique et cliquez sur "équiper ce skin".

## Technologies utilisée

Le jeu lui-même (client) est codé en Godot 4.4

Le serveur est codé en Python, avec MyHttpRequestHandler
